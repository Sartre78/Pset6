1.  According to its man page, what does getrusage do?
    
    getrusage returns usage data based on one of three
    possible int who's (RUSAGE_SELF, RUSAGE_CHILDREN,
    RUSAGE_THREAD), based on the struct rusage.

2.  Per that same man page, how many members are in a variable of type struct rusage?
    
    16 (although not all are required to be filled - kernel will set to zero).
